VRChatのアバター改変や制作を依頼するとき、最初に決めることが曖昧だとトラブルになりやすいです。
見た目の好みだけで進めると、Quest対応、表情、利用範囲、納品データで認識違いが起きます。

アバターの探し方、違法アバターの注意、アバター負荷の動画を見て整理すると、依頼前に必要なのは「かわいい見た目」だけではありません。
元モデルの権利、利用規約、Public化の可否、パフォーマンスランク、どこで使うアバターかまで決めておく必要があります。

## 依頼前に決めること

まずは、次の項目を言葉にします。

| 項目 | 決める内容 |
| --- | --- |
| 目的 | 自分用、配信用、イベント用、販売用ではないか |
| 予算 | 上限と追加費用の扱い |
| 納期 | いつまでに必要か |
| 対応環境 | PCのみか、Quest対応も必要か |
| 納品物 | Unitypackage、改変済みデータ、画像など |
| 利用規約 | 元アバターや衣装の規約で依頼改変が許可されているか |
| パフォーマンス | 普段使いか、撮影用か、イベント用か |

ここが曖昧なまま依頼すると、あとから「それも含まれると思っていた」が起きやすくなります。

## 依頼前に好みを固める

VRChatにはゲーム内のアバター検索ワールド、BOOTHやGumroadの販売アバター、VRoidからの変換、既存モデルの改変、フルスクラッチなど、いくつかの入口があります。
いきなりフルスクラッチ依頼へ行く前に、まずは自分の好みに近いアバターを探しておくと失敗しにくいです。

おすすめの順番はこれです。

1. パブリックアバターで好きな体型や雰囲気を探す
2. BOOTHなどで近いベースモデルを探す
3. 自分でアップロードできるか軽く試す
4. 自分では難しい改変だけを依頼する

「全部おまかせで作ってください」より、「このベースモデルに、この雰囲気の髪色、この用途で使いたい」と伝えた方が、制作側も判断しやすいです。

## Quest対応は先に確認する

VRChatのアバター依頼で特に確認したいのがQuest対応です。
PC向けの見た目をそのままQuestで再現できるとは限りません。

Quest対応では、シェーダー、ポリゴン数、マテリアル数、テクスチャ容量などに制限が出ます。
見た目をどこまで保つか、軽量化をどこまで行うかを先に相談しておくと安全です。

依頼文には、次のように書くと伝わりやすくなります。

```text
PC版での見た目を優先したいです。
Quest対応も可能ならお願いしたいですが、見た目が大きく変わる場合は事前に相談したいです。
```

## 権利と規約は必ず確認する

依頼でいちばん危ないのは、元モデルの権利が曖昧なまま進むことです。
動画では、ゲームデータのぶっこ抜き、無許可の版権アバター、販売アバターの無断パブリック化が問題として紹介されていました。

依頼前に確認するのはこのあたりです。

- 元アバターを自分が正規購入しているか
- 衣装や小物の改変依頼が規約で許可されているか
- 制作者にデータを渡してよい規約か
- 完成後にPublic設定にしてよいか
- 商用利用や配信利用に制限がないか

「パブリックワールドに置いてあるから使っていい」は危ないです。
販売アバターは、購入者本人がPrivateで使う前提のものが多いです。
依頼時に、購入していないデータを渡す、または受け取るのは避けます。

## 料金だけで選ばない

安い依頼が悪いわけではありません。
ただし、料金だけで選ぶと、修正回数、納品範囲、対応環境が合わないことがあります。

見るべきなのは、次の4つです。

- 過去の制作例が自分の好みに近いか
- 修正回数や追加料金が明記されているか
- Quest対応やアップロード補助の範囲が分かるか
- 納品後に自分で編集できる形か
- 規約や権利確認にきちんと触れてくれるか
- Very Poor回避や軽量化の方針があるか

「安いから依頼する」より、「自分の目的に合う範囲が明確だから依頼する」の方が失敗しにくいです。

また、SNSで向こうから突然「アバター作ります」と近づいてくる相手には慎重になります。
ポートフォリオ、過去実績、利用規約への理解、連絡先、支払い方法がはっきりしている相手を選びます。

## パフォーマンスランクも依頼条件に入れる

VRChatでは、アバターの重さが周囲の人の体験にも影響します。
動画では、カリングで見えなくしてもメモリに残ることがあり、Very Poorのアバター1体でも重さの原因になりうる、という話が出ていました。

だから依頼時には、見た目だけでなく「どこで使うか」も伝えます。

| 用途 | 依頼時の方針 |
| --- | --- |
| 普段使い | できれば軽め。Very Poor回避を相談 |
| イベント参加 | 表示負荷を優先して軽量化 |
| 撮影用 | 見た目優先。ただし用途を限定 |
| Quest対応 | 容量、PhysBone、マテリアル数を厳しめに確認 |

Very Poorが絶対悪というより、用途の問題です。
撮影用なら重くても許容される場合がありますが、イベントや普段使いなら軽さもマナーに近いです。

## 依頼文テンプレート

最初の連絡は、短くても必要情報が入っている方が返信されやすくなります。

```text
VRChatアバターの改変をお願いしたいです。
目的は自分用です。
希望内容は、色変更、髪型調整、表情追加です。
PC版を優先し、可能であればQuest対応も相談したいです。
予算は○円まで、希望納期は○月○日です。
参考画像と元アバターの情報を送れます。
元アバターと衣装は正規購入済みです。
利用規約上、改変依頼や制作者へのデータ共有が可能か確認中です。
普段使いしたいので、可能ならVery Poorを避けたいです。
```

テンプレートを使うと、聞かれる前に必要情報を出せます。
依頼はコミュニケーションなので、最初の整理がそのまま完成度につながります。

## 今日やること

今日依頼文を書く前に、この5つだけ確認します。

1. 好きなアバターの例を3つ集める
2. 元モデルと衣装の規約を開く
3. PCのみかQuest対応も必要か決める
4. 普段使いか撮影用か決める
5. 予算、納期、修正回数の希望を書く

これだけで、依頼の解像度がかなり上がります。
相手に丸投げするのではなく、判断材料を渡すことが、良いアバター依頼の第一歩です。
