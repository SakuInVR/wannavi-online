DTM初心者がいきなり1曲完成を目指すと、構成、音作り、ミックス、展開の全部で詰まりやすいです。
最初は4小節ループだけで十分です。

コード進行、ベース、ドラム打ち込みの解説動画を見て整理すると、4小節ループで大事なのは「センスで一気に作る」ことではありません。
ドラムの入力エリアを4小節で作る。コードを1小節ずつ置く。ベースはまずルート音にする。
この順番にすると、初心者でも音楽っぽい塊までたどり着けます。

## 4小節に入れるもの

最初のループには、次の5つを入れます。

| パート | 役割 |
| --- | --- |
| キックとスネア | リズムの骨組み |
| コード | 曲の雰囲気 |
| ベース | 低音の動き |
| 短いメロディ | 耳に残る要素 |
| 印象的な音 | 自分らしさやフック |

この5つが鳴るだけで、曲の種になります。
最初からイントロ、Aメロ、サビまで考えなくて大丈夫です。

## 先にドラムの入力画面を整える

ドラム打ち込み動画で具体的だったのは、音を置く前の準備です。
CubaseならGroove Agent SEを立ち上げ、GM MapでKickやSnareの名前を見えるようにし、スナップをONにして、鉛筆ツールで4小節の入力エリアを作る。

DAWは違っても、考え方は同じです。

- ドラム音源を1つ選ぶ
- Kick、Snare、Hi-hatの場所を分かる状態にする
- グリッドやスナップをONにする
- 4小節分の入力エリアを作る
- 1小節作ってコピーする

初心者がつまずくのは、音楽理論より先に「どこをクリックすれば音が入るのか」です。
だから最初は、音色探しより入力画面の安心感を優先します。

## 30分で作る手順

最初の10分でドラムを置きます。
キックは1拍目と3拍目、スネアは2拍目と4拍目に置くだけでも大丈夫です。
ハイハットは8分で刻むと、すぐにループ感が出ます。

次の10分でコードを4つ並べます。
コード動画では、難しい理論に入る前に、ルート音から半音を数えて三和音を作る考え方が紹介されていました。
たとえばメジャーコードなら、ルート音から4つ上、さらに3つ上に音を置きます。

最初は1小節に1コードでOKです。
音が飛びすぎてバラバラに聞こえるときは、コードの中の音を上下にずらして、近い位置に並べます。
これだけで、急にまとまりが出ます。

最後の10分でベースと短いメロディを足します。
ベースはコードの一番低い音、つまりルート音をなぞるだけで形になります。
慣れてきたら、8分音符で刻む、コードが変わる直前に食い込ませる、オクターブを混ぜる、という順番で変化をつけます。

メロディはスケール内の音から2音から4音だけ選びます。
長いメロディを作ろうとしなくてOKです。短いフックが1つあるだけで、ループはかなり曲っぽくなります。

## うまく聞こえないときの直し方

4小節がうまく聞こえないときは、音を増やす前に原因を分けます。

- リズムが弱いなら、キックとスネアの位置を見直す
- 暗すぎるなら、コードを変える
- 低音が動きすぎるなら、ベースをシンプルにする
- 印象が薄いなら、短いメロディを1つ足す

初心者は、足し算で解決しようとしがちです。
でも最初は、不要な音を減らして主役を見つける方がまとまりやすくなります。

特にベースは、動かしすぎると低音が濁ります。
まずはルート音だけ。次にリズム。最後にオクターブや経過音。
この順番だと、ベースが曲を支える感覚をつかみやすいです。

## ループができたら

次は、同じループから音を抜いたり足したりして8小節にします。
たとえば、前半4小節はドラムとコードだけ、後半4小節でベースとメロディを足すだけでも展開になります。

曲作りは、ゼロから全部を作るより、良いループを展開する方が続けやすいです。
まずは4小節を3つ作って、その中で一番気持ちいいものを伸ばしていきましょう。

## 今日やること

今日やるなら、この順番で十分です。

1. 4小節の空リージョンを作る
2. キック、スネア、ハイハットだけ置く
3. 1小節に1つずつコードを置く
4. ベースはルート音だけにする
5. メロディは2音から4音だけにする
6. ループ再生して、一番気持ちいいパートを1つ残す

ここまでできたら、もうDTMの最初の勝ちです。
完成曲ではなくても、自分の手で鳴るループができたなら、次の日に直す材料があります。
